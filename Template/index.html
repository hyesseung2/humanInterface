User Scenario 1 동네 근처에서 수선집을 확인

동네 근처에서 수선집을 확인하는 시나리오 입니다.

옷을 구매 했는데 옷이 잘 맞지 않아서 고민이다. 수선하여 내 몸에 잘 맞는 핏으로 잘 입고 싶다.

어플에 접속한 후 평가 및 리뷰를 확인하고 해당 지역의 전문 수선집 목록을 본다 리뷰를 확인하여 수선집을 선택하고 필요한 조정을 자세히 설명할 수 있는 기능으로 이동한다. 구체적인 내용(허리줄이기, 신체사이즈)을 포함해 바지의 현재 핏을 보여주는 사진을 올린다.

수선집을 선택하고 해당 사진, 신체 측정 정보를 보내면 예상 수선비와 예상 완료시간이 표기된다. 원하는 조건의 수선집을 발견하고 선택한다.

User Scenario 1-1 제품을 구매하는 곳에서 바로 수선

오프라인 매장의 경우 사는 곳에서 대부분 직접 수선을 진행하고 있으므로,
온라인 매장에서 제품을 사고 바로 수선하는 시나리오 입니다.

내 신체사이즈를 알고 있는데, 구매하고 싶은 옷의 바지 사이즈가 너무 길다. 딱 사고싶은 바지인데 수선집에 맡기러 가기는 귀찮다...

옷 구매사이트와 연동되어 있는 링크를 클릭하여 어플에 접속한 후 수선집 목록을 확인 한 후에 해당 수선집과 연결하여 구매하려던 옷을 바로 수선집으로 보낸다.

수선집들을 방문하여 맡길 일 없이 수선된 옷을 바로 받아볼 수 있어 시간낭비가 없어 편했다.

User Scenario 2 수선 픽업 배송

가지러가기 불편한 유저들의 불편함을 해소하는 시나리오입니다.

최근 일이 너무 많아져서 수선한 제품을 출퇴근 시간에 수선집에다가 맡기거나 픽업을 하러 가기가 어렵다.

앱에서 수선집에다가 수거 요청을 한 다음 근처 편의점이나 공유 박스에서 맡기고 픽업을 진행한다.

일이 너무 많아서 공유박스나 편의점에 맡기기도 귀찮을때, 추가 요금을 내고 집앞까지 편하게 배민 수거/배달을 받았다. 가지러 가야하는 불편함이 해소되었다.

User Scenario 3 수선 애프터 서비스

수선 이후 잘못된 수선의 불편함을 해소해주는 시나리오입니다.

수선이 잘못 되어서 속상하다. 옷을 버릴수도 없고 어떻게 처리를 해야할지 모르겠다.

앱 내역에 제시된 요구사항 정의가 제대로 됐는지 확인 후에 재수선 요청을 진행한다.

재수선 요청이 번거로울 경우, 앱에 연동된 기부 선택을 통하여 소득공제 혜택을 받아 만족감을 얻는다.
